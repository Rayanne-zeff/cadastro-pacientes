# Cadastro de Pacientes

## Tecnologias utilizadas
```
Spring Boot
Spring Framework
Spring JPA
Spring Security
Spring Security Oauth2
Lombok
PostgreSQL
```

## Instalação
```
mvn install
```

## Base de dados
#### Para importar a base de dados com dados basta utilizar o arquivo abaixo do diretório:
```
src/main/resources/database/cadastro_pacientes.sql
```

## Executando
```
spring-boot:run
```
